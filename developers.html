Build your own iobio app
(coming soon)
We are working hard on a developer release that will include interactive genomic charts, javascript libraries for managing analysis and websocket connections, and the ability to add new analyses. If you'd like know when this comes out sign up below.
We have an early access program. If you'd like to work with us to build your own apps before the developer toolkit is released, email us (iobioproject@gmail.com) with your ideas.
Install iobio locally
You can install iobio (the apps and supporting web services) locally. This way none of your data leaves your organizations network (often a requirement to analyze private patient data). We have hospitals running local instances. Please refer to the install guide. This is currently a multi-step process and we are working on making it easier.
